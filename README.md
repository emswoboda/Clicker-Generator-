# Clicker Generator

A browser-based tool for turning an uploaded image into a custom emoji clicker design.

Goal:
- Upload a simple image
- Trace the outside silhouette
- Preview the clicker shape
- Eventually export cap and base STL files for 3D printing

## Planned Features

- Image upload
- Automatic silhouette tracing
- Matching top cap and base shape
- Keyboard switch clicker dimensions
- STL export# Clicker-Generator-
